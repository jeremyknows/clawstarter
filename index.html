Questions?

Is this safe?

Yes. OpenClaw runs in a sandboxed Mac user account with zero access to your personal files, browser history, or passwords. It can only do what you explicitly grant permission for.

The setup guide walks you through security best practices: file permissions, firewall settings, and disk encryption. Your API keys are stored locally and never transmitted anywhere except to the AI provider you choose.

What does it cost?

OpenClaw is free. The software costs nothing.

You pay only for the AI model usage (like paying for electricity). Budget options start at $5-15/month for light use. You set spending limits during setup, so there are no surprise bills.

Think of it like this: OpenClaw is the car (free), the AI provider is the gas station (pay-as-you-go).

Do I need to know how to code?

Nope. If you can copy and paste text into Terminal, you can set up OpenClaw.

The setup guide is written for non-technical users. Every command is explained in plain English. If you get stuck, the FAQ and troubleshooting sections have your back.

What can it actually do?

Out of the box, your AI can:

  • Answer questions and have conversations
  • Search the web for real-time info
  • Remember context from past chats
  • Work across multiple messaging apps

You can add skills later: calendar management, email triage, file organization, camera access, and more. Each skill is optional and explained in the Foundation Playbook.

What's the catch?

There isn't one. This is an open-source project built for real people who want a personal AI without vendor lock-in.

You own your data. You control the costs. You decide what it can access. No subscriptions, no upsells, no tracking.

My Mac goes to sleep. Will this work?

The setup guide includes a command to keep your Mac awake (but turns off the screen to save power). This ensures your AI is always responsive.

If you need your Mac to sleep (like a laptop you carry around), you can skip that step‚Äîjust know your AI won't respond while the Mac is asleep.

Can I use this on multiple Macs?

Yes! Each Mac runs its own instance of OpenClaw. They don't sync automatically, but you can manually share configuration files or memory snapshots between them.

What if I want to uninstall it?

Delete the bot user account from System Settings ‚Üí Users & Groups. That's it. Everything is contained in that one user account.
